AlphaTeX is a LaTeX package allowing for calculation and plot generation inside LaTeX documents. 


# Commands

###### \calc{}

This command evaluates a mathematical expression, taking Wolfram Language syntax. For example 

```\calc{Exp[I [\Pi]]}```
